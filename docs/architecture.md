# 企业办公系统架构设计

## 系统名称
系统名称可由管理员在后台配置，默认名称为"智慧办公系统"

## 技术架构

### 前端架构
- **框架**: React 18 + TypeScript
- **UI组件库**: Ant Design 5
- **样式**: TailwindCSS + CSS Modules
- **状态管理**: Zustand
- **路由**: React Router v6
- **HTTP客户端**: Axios
- **实时通信**: Socket.io-client

### 后端架构
- **运行时**: Node.js 20
- **框架**: Express.js
- **语言**: TypeScript
- **ORM**: Prisma
- **数据库**: PostgreSQL
- **认证**: JWT (jsonwebtoken)
- **密码加密**: bcrypt
- **文件上传**: Multer
- **实时通信**: Socket.io
- **API文档**: Swagger

### 数据库设计
- **类型**: PostgreSQL
- **ORM**: Prisma
- **连接池**: 通过Prisma管理

## 模块设计

### 1. 用户权限模块 (RBAC)
- 用户管理
- 角色管理
- 权限管理
- 组织架构

### 2. 通讯录模块
- 部门管理
- 员工信息管理
- 组织架构树
- 权限控制

### 3. 即时通讯模块
- 单聊
- 群聊
- 消息类型：文字、图片、文件
- @功能
- 消息已读回执

### 4. 审批OA模块
- 流程设计器（可视化）
- 表单设计器
- 审批流转
- 审批历史

### 5. 公示公告模块
- 公告发布
- 公告分类
- 阅读统计
- 置顶功能

### 6. 财务报销模块
- 报销申请
- 费用类型
- 审批流程
- 报销统计

### 7. 客户管理模块 (CRM)
- 客户信息
- 跟进记录
- 合同管理
- 商机管理

### 8. 门户页面
- 信息展示
- 快捷链接
- 待办事项
- 公告轮播

## 性能目标
- API响应时间 < 5ms（本地测试环境）
- 页面首屏加载 < 2s
- 支持并发用户 100+

## 部署架构

### 免费方案
```
┌─────────────────────────────────────────────────────────┐
│                      用户浏览器                           │
└─────────────────────┬───────────────────────────────────┘
                      │ HTTPS
┌─────────────────────▼───────────────────────────────────┐
│                   Vercel (前端)                          │
│              React SPA 静态托管                          │
└─────────────────────┬───────────────────────────────────┘
                      │ API请求
┌─────────────────────▼───────────────────────────────────┐
│                  Railway/Render (后端)                   │
│              Node.js + Express API                       │
└─────────────────────┬───────────────────────────────────┘
                      │ 数据库连接
┌─────────────────────▼───────────────────────────────────┐
│                  Supabase (数据库)                       │
│              PostgreSQL + 文件存储                       │
└─────────────────────────────────────────────────────────┘
```

## 安全设计
- JWT认证 + Refresh Token
- 密码bcrypt加密
- API请求限流
- SQL注入防护 (Prisma ORM)
- XSS防护
- CORS配置
